## Points clés

### 1. [ Le fichier json ]

Il fait le lien, dans le but de faire l'échange de données entre le navigateur web et le serveur. Dans notre cas, le logiciel Visuel Studio Code qui a un serveur local intégrer avec "Live Server".



### 2. [ API ]

Une API aide a échanger des données entre deux interfaces logicielle passant par le web. Une API est une sources de données énorme, qui contient beaucoup d'informations. Par exemple, l'API REST Countries, qui regroupe tous les pays du monde avec leur capital et le nombre d'habitant.



### 3. [ La commande aria-live="polite" ]

Cette commande informe aux lecteurs d'écran l'état du chargement de la demande pour afficher l'information. C'est très pratique pour savoir si ca marche ou non.
