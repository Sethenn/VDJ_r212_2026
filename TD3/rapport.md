## Points clés

### 1. [ Expliquez le modèle événement → état → rendu avec un exemple de votre choix (pas un exemple du sujet) ]

Le modèle événement → état → rendu est les étapes que fait le JavaScript pour réaliser la tâche. 

Un exemple : 
Situation : Je like une publication sur Instagram.
Evenement : Clic sur le bouton like.
Etat : publication: 65 likes → 66 likes.
Rendu : Le compteur des likes augmente de 1.


### 2. [ Expliquez en quoi classList.toggle est la passerelle entre JS et CSS ]

La commande classList.toggle sert à ajouter si un fichier est absent ou le retirer s'il est présent. Le JS ajoute/retire des classes et le CSS définit ce que ces classes changent visuellement. Cette commande fait la pareselle entre les deux.
